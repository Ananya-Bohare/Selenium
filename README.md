# Selenium Cucumber BDD Automation Framework

## Overview

A Selenium automation framework developed using Java, Maven, Cucumber (BDD), JUnit 4, and the Page Object Model.

## Tech Stack

- Java
- Selenium WebDriver
- Cucumber BDD
- JUnit
- Maven
- Page Object Model (POM)
- Eclipse IDE

## Features

- BDD Feature Files
- Scenario Outline
- Data-driven testing
- Page Object Model
- Page Factory implementation
- HTML, JSON and XML Reports
- Maven Build Support
